# Speech-to-Text Application

A web-based Speech-to-Text application that records audio from the user's microphone and converts speech into text using the Deepgram API.

## Features

- Record voice from microphone
- Convert speech to text
- Copy transcript
- Download transcript
- Transcript history
- Error handling
- Responsive UI

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS

### Backend
- Flask
- Python

### API
- Deepgram Speech-to-Text API

### Deployment
- Frontend: Vercel
- Backend: Render

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
